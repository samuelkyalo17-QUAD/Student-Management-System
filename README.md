#=#Student Management System (SQL Project)

#=# Overview
This project is a relational database system for managing students, courses, lecturers, and enrolments.

#=# Technologies Used
- MySQL
- SQL (DDL & DML)

#=# Files Included
- schema.sql → Database structure (tables & relationships)
- data.sql → Sample data inserts
- queries.sql → Analytical queries (JOINs, GPA, performance)

#=# Features
- Student-course enrolment system
- Foreign key relationships
- GPA calculation using SQL
- Course performance analysis
- Student performance tracking

#=#Skills Demonstrated
- Database design
- SQL joins
- Aggregation functions (AVG, COUNT)
- Data analysis using SQL

#=# How to Use
1. Run schema.sql first
2. Run data.sql to insert data
3. Run queries.sql to test analysis queries
